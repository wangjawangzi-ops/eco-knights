# Eco-Audit Campus Calculator 🌱

This is a **college-level carbon footprint calculator** for departments.  

### Features
- Students/staff can enter daily usage data.
- Calculates CO₂ emissions based on plastic bottles, AC hours, and pages printed.
- Displays a live leaderboard comparing departments (lower CO₂ wins!).
- Built with Python + Streamlit.

### How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
